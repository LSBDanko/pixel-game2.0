# 像素冒险游戏网站

这是一个基于HTML、CSS和JavaScript开发的像素风格小游戏网站。

## 功能特点

- **首页**: 展示游戏标题和主要功能入口
- **游戏开始界面**: 选择角色和难度，开始游戏
- **MVP界面**: 显示玩家排行榜
- **数据显示界面**: 展示游戏统计数据

## 游戏玩法

1. 在首页点击"开始游戏"按钮
2. 在游戏开始界面选择角色和难度
3. 点击"开始冒险"按钮开始游戏
4. 使用键盘方向键控制角色移动
5. 收集彩色方块获取分数
6. 避开红色敌人，否则游戏结束

1. 直接在浏览器中打开index.html文件即可运行
2. 无需安装任何依赖或服务器
